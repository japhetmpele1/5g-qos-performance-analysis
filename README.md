# 5g-qos-performance-analysis
This project analyzes the Quality of Service (QoS) performance of a simulated 5G network environment. It focuses on key performance indicators such as latency, throughput, packet loss and availability across different 5G service categories.

## Objectives
- Generate realistic 5G QoS datasets
- Evaluate network performance by service type
- Compare QoS indicators across deployment zones
- Visualize results using basic statistical plots

## Service Types
- URLLC: Ultra-Reliable Low-Latency Communications
- eMBB: Enhanced Mobile Broadband
- mMTC: Massive Machine-Type Communications

## Technologies
- Python
- Pandas
- Matplotlib

## Outputs
- CSV summaries
- Latency distribution by service type
- Throughput comparison by zone
- Packet loss comparison by service type

## Research Relevance
This project is relevant to:
- 5G communication systems
- QoS evaluation
- Intelligent transport systems
- Reliability and resilience of wireless communications
